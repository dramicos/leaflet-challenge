# Overview:
This is a Leaflet map that displays all eathquakes that have happened in the last 7 days as recorded by the USGS.   


## Data Sets:

* Link to the USGS earthquake feed geojson
    * Link: "https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/all_week.geojson"
    * Description: Only the 'Electric and Alternative Fuel Charging Stations.csv' from the Data Explorer at the bottom of the page.

* Data on tectonic plates
	* Link: https://github.com/fraxen/tectonicplates
    * Description: A github repo that contains compiled usgs data in json format
